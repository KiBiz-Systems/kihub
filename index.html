/02 — Flagship Project

KiBuild MCP.
Structured access.

A FileMaker-aware MCP server that gives AI coding tools deep, structured access to your FileMaker schema — scripts, layouts, tables, relationships, and dependencies.

KiBuild MCP

Works with Claude Code, Cursor, Windsurf, Codex, Antigravity (Google), and any MCP-compatible AI tool.

v0.2.0 MIT License Go 1.21+

Without KiBuild (Grep / Raw XML)

  • ✕Find where a script is called: grep -r "Send Email" across hundreds of XML files — noisy, slow, misses numeric IDs.
  • ✕Find where a field is used: Manual regex over calculation XML — misses layouts and relationships entirely.
  • ✕Understand a script: Read hundreds of lines of raw XML step-by-step.
  • ✕Trace a relationship: Open the relationship graph XML and parse join predicates manually.
  • ✕Search the whole schema: Grep tokenizes everything — expensive, context-filling, slow.

With KiBuild MCP

  • →find_script_references_in_scripts returns exact call sites with file paths and line numbers.
  • →find_field_references in calculations, layouts, and relationships in one call.
  • →find_script returns a clean step list with names resolved.
  • →inspect_relationships returns structured JSON immediately.
  • →search_index searches workspace_map.md — token-efficient, instant.

/04 — Tool Reference

32 tools.
4 disciplines.

KiBuild MCP exposes 32 tools across four categories. No fluff, no overlapping utilities. Each tool solves a specific FileMaker navigation or impact-analysis problem.

/01

Schema navigation

Find scripts, layouts, and tables by name. Inspect relationships. Keyword search the workspace_map.md index. Generate or refresh the schema map. The foundational lookup layer.

find_script find_layout find_table inspect_relationships search_index generate_schema_map
/02

Impact analysis

Trace where a script is called, where a field is used, or what references a table occurrence. Covers scripts, layouts, calculations, relationships, value lists, and variables.

find_script_references find_field_references find_variable_references find_layout_references find_to_references find_relationship_predicates
/03

XML & generation

Explode exports, extract steps, resolve IDs, trace dependencies, match revisions. Validate FMXML snippets and WebViewer HTML. Write versioned artifacts to the outbox.

explode_xml_export xml_extract_steps xml_lookup_name validate_fmxmlsnippet write_outbox_artifact
/04

Specialist skills

Load domain-specific expertise into AI context. Pro scriptwriter mode, script analysis, FM XML serializer, and script debugging. Skills shape how the AI applies the other 31 tools.

load_skill pro_scriptwriter script_analysis fm_xml_serializer script_debug
/05 — Get Started

KiBuild MCP.

One binary. Zero dependencies. Install KiBuild MCP in seconds. The installer handles the PATH, the config, and the AI guides automatically.

Install with a single command — then run /setup-kibuild inside Claude Code

$ curl -fsSL https://raw.githubusercontent.com/KiBiz-Systems/kibuild-mcp/main/install.sh | sh |
/01 — Download & Install

Install the binary

Run the interactive installer to download the binary, configure your PATH, and get the helper commands.

macOS / Linux
$ curl -fsSL https://raw.githubusercontent.com/KiBiz-Systems/kibuild-mcp/main/install.sh | sh
Windows (PowerShell)
> irm https://raw.githubusercontent.com/KiBiz-Systems/kibuild-mcp/main/install.ps1 | iex
Verify with: kibuild-mcp --version
/02 — Configure MCP

Run the setup wizard

Configure the MCP server for your AI environment (Claude Code, Cursor, Windsurf, Codex, etc.).

Terminal (Interactive)
$ kibuild-mcp --setup
Claude Code Command
> /setup-kibuild
Writes configurations automatically.
/03 — Export XML

Export your schema

Save a copy of your FileMaker schema as XML. In FM 2026+, export specific catalogs to speed up processing.

FileMaker Script Step
# Save database as XML:
Save a Copy as XML [
  Window Name: "MySolution";
  Destination: "export.xml"
]
Or use Tools → Save a Copy as XML...
/04 — Initialize & Run

Connect your project

Initialize the workspace guides in your project directory so your AI knows how to invoke the MCP server.

Claude Code Command

Verify MCP Server
